# PUBLISH_CHECKLIST.md

## Purpose

Pre-publish checklist for the Kentucky County Map public surface.

Use this before publishing, sharing, embedding, or presenting the map as an ABLE-KY-aligned civic/legal-access navigation tool.

## Current public copy source

`public/INDEX_COPY.md`

## Publish posture

KY County Map may publish as an ABLE-KY-aligned county navigation tool.

It may carry:

- Kentucky county map interface files
- county-level navigation links
- ABLE-KY-facing resource routing
- static HTML/CSS/JS map code
- accessibility improvements
- mobile usability improvements
- embed instructions

It should not become:

- Perspection governance
- Faceless Reclaimation testimony
- Trinity Framework site
- System Analysis Reference
- raw legal evidence archive
- public accusation page
- catch-all ecosystem homepage

## Pre-publish checklist

Before publishing, confirm:

- [ ] The page identifies itself as Kentucky County Map or KY County Map.
- [ ] The page clearly frames the map as navigation, not legal advice.
- [ ] ABLE-KY framing is accurate and approved before public organizational use.
- [ ] The map loads on mobile.
- [ ] County labels, hover states, or markers are readable.
- [ ] Active counties are visually clear.
- [ ] Inactive counties are visually clear.
- [ ] Click targets route correctly.
- [ ] Resource URLs are current.
- [ ] No county routes to a placeholder unless clearly labeled as unavailable or pending.
- [ ] The map does not determine eligibility.
- [ ] The map does not replace direct contact with the relevant organization or resource provider.
- [ ] Related projects are labeled as related, not absorbed.

## Voice check

The page may be:

- simple
- civic
- navigational
- accessible
- county-specific
- practical
- low-friction

The page should avoid becoming:

- legal advice
- testimony-heavy
- governance-heavy
- ecosystem-heavy
- over-explained
- accusation-based
- a general evidence archive

## Boundary check

Allowed line:

> KY County Map is an ABLE-KY-aligned interactive Kentucky county navigation tool for routing people toward county-level civic, legal-access, and rights-restoration resources.

Allowed line:

> This map is a navigation tool. It does not give legal advice, determine eligibility, or replace direct contact with the relevant organization or resource provider.

Avoid lines like:

> This map provides legal advice.
> This proves rights violations.
> This is Perspection.
> This is Faceless Reclaimation.
> This is the entire ecosystem.
> This determines whether you qualify.

## Publish-safe description

KY County Map is an ABLE-KY-aligned interactive Kentucky county navigation tool for routing people toward county-level civic, legal-access, and rights-restoration resources.

## Ultra-short description

County-level direction for Kentucky access routes.

## Final pass questions

Before publishing, ask:

1. Does the map load and work on mobile?
2. Are county routes correct and current?
3. Is the legal-advice boundary clear?
4. Is ABLE-KY framing accurate and approved for public use?
5. Are related projects kept separate?
6. Can a visitor understand what to click without needing the whole ecosystem explained?

## Operating rule

This repository is a navigation tool.

It helps people find the right county-level direction.
It does not absorb the full ecosystem.
It does not become governance, testimony, or analysis by default.
