# PROJECT_BOUNDARY.md

## Project identity

This repository is an ABLE-KY-aligned Kentucky county navigation map tool.

It is designed to support county-level resource navigation, civic access, rights-restoration pathways, and public-facing regional orientation for Kentucky communities.

## This project is not

- It is not the canonical Perspection governance repository.
- It is not the Faceless Reclaimation public/story container.
- It is not the Trinity Framework site.
- It is not the Living Survival Guide field-tool container.
- It is not a general evidence archive or legal accusation repository.

## Allowed material

- Kentucky county map interface files
- County-level resource navigation links
- ABLE-KY-facing civic/legal-access routing
- Static HTML/CSS/JS map code
- Accessibility and mobile usability improvements
- Documentation for embedding or adapting the map
- Clearly labeled Trinity Framework references when used as structural method, not identity

## Not allowed material

- Perspection governance rules as if this repo owns them
- Faceless Reclaimation story material as the main identity
- General legal case files or personal evidence dumps
- Unlabeled Manus exports from unrelated projects
- Copy that turns this map into a doctrine, testimony archive, or catch-all ecosystem hub

## Related but separate

- `7u4n3r/Perspection` is the canonical governance/reference surface.
- `7u4n3r/faceless-reclaimation` may hold Reclaimation-facing public material.
- `7u4n3r/trinity-framework-site` is reserved for Trinity-specific site material.
- `7u4n3r/Trinity-tools` may hold working tool builds and implementation experiments.
- `7u4n3r/system-analysis-reference` may hold neutral analysis/reference material.

## Operating rule

This repository is a navigation tool.

It helps people find the right county-level direction.
It does not absorb the full ecosystem.
It does not become governance, testimony, or analysis by default.
